Trust & Trace

Alternance– Worldline, 2025

Présentation du Projet

Trust & Trust est une application web de revente d’articles de seconde main, s’inspirant du modèle de plateformes comme Vinted, mais avec une plus-value majeure : la garantie d’authenticité. Chaque produit est soumis à une vérification rigoureuse avant sa mise en ligne grâce à des outils d’analyse intelligente.

Fonctionnalités phares

  • Capture d’image du produit pour comparaison automatique avec les visuels officiels (scraping et matching).
  • Analyse de code barre / numéro de série pour vérifier l’authenticité du produit en ligne.
  • Interface utilisateur fluide pour publication, messagerie et gestion de profils.
  • Système de scoring de confiance basé sur les vérifications passées.

Responsabilités techniques

  • Conception de l’interface avec React.js et gestion dynamique des formulaires de dépôt.
  • Implémentation du backend en Java Spring Boot pour gérer les flux de vérification et les échanges entre services.
  • Utilisation de scraping intelligent et intégration d’APIs tierces pour croiser les informations des produits.
  • Déploiement de l’application sur Google Cloud Platform (GCP), avec pipeline CI/CD.

Stack technique

  • React.js
  • Spring Boot (Java)
  • Google Cloud Platform (App Engine, Cloud Functions)
  • Scraping Web, APIs de vérification produits
  • MongoDB (ou PostgreSQL selon la version déployée)

Résultats & Apports

Ce projet m’a permis de développer une approche complète d’un produit web complexe, mêlant UI, sécurité, et automatisation. J’ai acquis une solide expérience sur la fiabilisation des données utilisateurs et la gestion cloud. En bonus, j’ai pu proposer des améliorations UX comme un indicateur de fiabilité dynamique affiché sur chaque annonce.
