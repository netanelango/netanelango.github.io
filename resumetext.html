ResumeText

Projet personnel – 2024

Présentation

ResumeText est une application Angular qui génère automatiquement un document PDF de synthèse à partir des réponses saisies dans un formulaire. L’objectif est de permettre aux utilisateurs de produire rapidement un compte rendu clair et structuré, utilisable dans des contextes professionnels, scolaires ou administratifs.

Fonctionnalités

  • Formulaire dynamique avec différents types de champs (textes, dates, options...)
  • Génération de synthèse personnalisée à partir des réponses.
  • Export automatique au format PDF via une bibliothèque Angular.
  • Possibilité de personnaliser l'en-tête, le pied de page et la mise en page du document.

Mon rôle

  • Conception de l'interface utilisateur responsive avec Angular.
  • Création d’un service de traitement de données pour la synthèse automatique.
  • Intégration de la génération PDF (via jsPDF ou pdfMake).
  • Amélioration de l’expérience utilisateur (feedback visuel, validations, prévisualisation PDF).

Technologies utilisées

  • Angular
  • TypeScript
  • jsPDF / pdfMake
  • HTML5 / CSS3

Ce que j’ai appris

Ce projet m’a permis de mieux comprendre la manipulation dynamique de données côté front-end, ainsi que l’intégration de services de génération de fichiers PDF dans une application Angular. J’ai également renforcé mes compétences en structuration de formulaires complexes.
